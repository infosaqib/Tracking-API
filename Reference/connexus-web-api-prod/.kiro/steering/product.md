# Product Overview

Connexus is an internal tool platform that provides APIs for property management and vendor services. The system manages relationships between clients, vendors, properties, and services through a comprehensive web API.

## Core Domains

- **Client Management**: Commercial, HOA, hotel, multi-family, retail, and student housing clients
- **Vendor Management**: Service providers with approval workflows and serviceable areas
- **Property Management**: Property portfolios with contacts, services, and geographic data
- **Service Management**: Service categories, contracts, and scope of work (SOW) management
- **RFP/RFQ System**: Request for proposals and quotes with document management
- **Background Jobs**: Asynchronous processing for exports, document generation, and data processing
- **User Management**: Multi-tenant user system with role-based permissions via AWS Cognito

## Key Features

- Multi-tenant architecture supporting different client types
- Document generation and export capabilities
- Geographic data management with PostGIS support
- AI-powered contract data extraction
- Comprehensive audit trails and soft deletes
- Real-time health monitoring and metrics
